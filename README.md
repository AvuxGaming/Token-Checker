# Token-Checker
## Installing Package :
```console
$ npm install
```
***
## Usage :
1. Open file `tokens.txt` and enter your tokens (1 token - 1 line).
2. `main.js` and `tokens.txt` must be in the same folder.
3. Run `main.js`.
```console
$ node main.js
```
***
## Config :
Config Module
```console
{
  "interval": 500,
  "usernames": false
}
```
* Result With Username : 
```
  "usernames": true
```
* Result Without Username : 
```
  "usernames": false
```
